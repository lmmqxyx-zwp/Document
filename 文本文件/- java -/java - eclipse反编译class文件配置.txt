1.下载jad.exe文件
2.下载net.sf.jadclipse_3.3.0.jar文件

1、配置eclipse
	新建eclipse/jad文件夹
	- eclipse\jad\temp\
	- eclipse\jad\jad.exe

2.把下载好的net.sf.jadclipse_3.3.0.jar文件放置在eclipse文件夹下面的/plugins

3.打开你的eclipse软件
